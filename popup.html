Время, на которое расширение отключается после запуска браузера.

Нужно, чтобы отключить автоматическое перемещение вкладок, пока браузер открывает ранее закрытые вкладки

wtf?

Создать группу вкладок arc-tabs. Она будет содержать одну пустую вкладку, которая будет использоваться для скролла наверх панели вкладок при открытии новой вкладки.

При удалении группы она будет создана снова автоматически.

Чтобы удалить группу arc-tabs - можно выключить расширение, удалить группу и не создавать ее снова через эту кнопку
